# Push_swap

## 📌 Overview

Push_swap is an algorithmic project focused on sorting data using a limited set of operations and two stacks.

---

## ⚙️ Features

* Sorting using two stacks (A and B)
* Limited instruction set (push, swap, rotate)
* Optimized algorithms for minimal operations

---

## 🧠 Key Concepts

* Sorting algorithms
* Data structures (stacks)
* Algorithm optimization
* Complexity analysis

---

## 🛠 Tech Stack

* C
* Linux

---

## ▶️ Usage

```bash
make
./push_swap 3 2 1 5 4
```
